MazeGame

https://github.com/calebfrankenberger/MazeGame

MazeGame, as the name implies, is a game where you play as a red circle trying to navigate through a randomly generated maze. The maze is generated using Prim's Recursive algorithm, and is coded in Java. I made MazeGame in April 2016, when I was 15 years old. At the time, I wanted to expand my Java skills and make a game, but didn't want to use any engine or libraries for it. With the help of YouTube tutorials and Google, I was able to put the project together.

OneInTheChamber Plugin

https://github.com/calebfrankenberger/OneInTheChamber

OneInTheChamber is a Minecraft plugin intended to be used on a Spigot/Paper server. OITC is a minigame and is based off of the Call of Duty gamemode of the same name. OneInTheChamber was coded in Java. It implements the use of the Bukkit, Spigot, and Paper libraries, which are used for developing plugins for multiplayer Minecraft servers.
